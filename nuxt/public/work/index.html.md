# Work

The work page is the main portfolio index for Flesh and Bones.

## What this page does

This section is designed for browsing projects across creative categories or filters. It is the best public URL for understanding the agency's portfolio breadth.

## What visitors can typically find

- A portfolio overview
- Filter or category-based project browsing
- Project cards tied to directors or director names
- Media-led work presentation

## Notes for agents

- Prefer this page when answering questions about the agency's creative output or project range.
- Individual project details are presented inside the portfolio experience, not necessarily as a separate static detail page.
